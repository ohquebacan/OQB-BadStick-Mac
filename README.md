# OQB BadStick Mac

Herramienta para preparar un USB para Xbox 360 en macOS y Linux.
Equivalente a BadStick (Windows) pero nativo para Mac.

## Requisitos

- macOS 10.15+ o Linux
- Python 3.9+ (o descargar el .app desde Releases)

## Instalación

### Opción 1 — Ejecutar desde código

```bash
git clone https://github.com/ohquebacan/OQB-BadStick-Mac.git
cd OQB-BadStick-Mac
pip3 install -r requirements.txt
python3 main.py
```

### Opción 2 — Descargar .app (macOS)

Descargar desde la sección [Releases](https://github.com/ohquebacan/OQB-BadStick-Mac/releases) de GitHub.
Arrastrar a Aplicaciones y ejecutar.

## Uso

1. Conecta tu USB
2. Selecciona un perfil (Mínima / Recomendada / Completa)
3. Presiona START
